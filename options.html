CF Access Header Injector

Configure one Cloudflare Access service token per HTTPS site. Every enabled site injects only CF-Access-Client-Id and CF-Access-Client-Secret into its own requests.

Security note

Client Secrets stay in this browser profile's local extension storage and are never synchronized. Browser storage is not a hardware-backed secret vault: use a dedicated, least-privilege service token for each site, keep every scope narrow, and revoke a token if this browser profile is compromised.

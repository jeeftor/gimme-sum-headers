Security warning

Header values and their local request rules stay in this browser profile and are never synchronized. This is not a password manager or hardware-backed secret vault. They are sent only to HTTPS sites you explicitly map.

Prefer dedicated, least-privilege credentials and exact hosts. A wildcard sends its header set to every matching subdomain. Revoke affected credentials if this browser profile or device is compromised.
